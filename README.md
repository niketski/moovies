A movie search app built with React Js and TMDB api.

[Demo](https://niketski.github.io/moovies/)

## Installation 

1. Clone the repo 
> git clone https://github.com/niketski/moovies.git

2. Install NPM packages.
> npm install

3. Run the app.
> npm start
